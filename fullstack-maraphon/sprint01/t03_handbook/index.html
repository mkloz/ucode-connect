HTML

Description

HTML is the standard markup language for creating web pages.

Code Example

<html>
  <head>
    <title>Title</title>
  </head>
  <body>
    <h1>Heading</h1>
    <p>Paragraph</p>
  </body>
</html>

CSS

Description

CSS is a stylesheet language used for describing the presentation of a document written in HTML.

Code Example

body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
}

Protocols

Description

Protocols are a set of rules that govern the exchange of data over a network.

HTTP/HTTPS

HTTP (Hypertext Transfer Protocol) and HTTPS (HTTP Secure) are protocols used for transferring data on the World Wide Web.

DNS

DNS (Domain Name System) is a protocol for translating domain names into IP addresses.

IP

IP (Internet Protocol) is the principal communications protocol in the Internet protocol suite for relaying datagrams across network boundaries.
